# PRO Homework: User Manager and Custom Exceptions

Это задание реализует систему управления пользователями с использованием собственных типов исключений и обработкой ошибок в сложных сценариях.

## Структура проекта

- pro_users.py — основная логика программы:
  - модель пользователя `User`
  - менеджер пользователей `UserManager`
  - пользовательские исключения
  - тестовые сценарии

## Классы

### User
Модель пользователя с атрибутами:
- username — имя пользователя
- email — email
- age — возраст

Методы:
- __init__ — инициализация данных
- __str__ — строковое представление

## Менеджер пользователей: UserManager

Методы:

- add_user(user)  
  Добавляет нового пользователя.  
  Если имя уже занято — выбрасывает `UserAlreadyExistsError`.

- remove_user(username)  
  Удаляет пользователя по имени.  
  Если пользователь не найден — выбрасывает `UserNotFoundError`.

- find_user(username)  
  Ищет пользователя.  
  Если отсутствует — выбрасывает `UserNotFoundError`.

## Пользовательские исключения

- UserAlreadyExistsError  
  Выбрасывается при попытке повторного добавления пользователя с существующим именем.

- UserNotFoundError  
  Выбрасывается при поиске или удалении несуществующего пользователя.

## Примеры запуска

Запуск программы:

python pro_users.py


В результате выполняются тестовые сценарии:
- добавление пользователей
- обработка попытки добавить пользователя повторно
- удаление существующего и несуществующего пользователя
- поиск пользователя и обработка ошибок
